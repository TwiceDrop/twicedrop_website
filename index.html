UP主B站动态推送插件

UP主B站动态推送插件，是基于TRSS(TimeRainStarSky)Yunzai开发的一款可以实时监控指定UP主的动态并实时推送到指定群聊的QQ机器人插件，功能：

按群设置需要检测的 UP 主 B 站 UID
管理本群检测 UID 列表
只有主人可以修改推送设置
支持推送时 @全体成员
支持推送时 @指定成员
推送动态标题、封面、UP 主名称和动态地址

UP主B站催播插件

UP主B站催播插件，是基于TRSS(TimeRainStarSky)Yunzai开发的一款可以实时监控指定UP主的开播情况并定时提醒UP主开播的QQ机器人插件，功能：

按群设置需要监控的 UP 主 B 站 UID
为每个 UID 设置需要 @ 的 QQ 号，提醒 TA 去开播
支持为每个 UID 设置监控时间段
支持设置提醒间隔，例如时间段内每 1 小时提醒一次
支持编辑未开播提醒内容
定时获取 UP 主直播状态，只在未开播且处于监控时间段内提醒
与 UP主B站动态推送插件 共用 B 站 Cookie 配置

请注意，该插件需要依赖UP主B站动态推送插件

Rammap内存清理自动化

1. 检查名为 `\RAMMap Auto Clean` 的计划任务是否已存在；如果存在，会弹出 Windows 提示框，并在控制台暂停等待按键，不覆盖原任务。
2. 自动申请管理员权限。
3. 弹出文件选择框，由你选择 `RAMMap.exe` 或 `RAMMap64.exe`。选择框会默认定位到脚本所在目录，并预选根目录中的 `RAMMap.exe`。
4. 要求输入清理间隔，允许范围为 1–1439 分钟。
5. 创建以 `SYSTEM` 账户、最高权限运行的 Windows 计划任务。
